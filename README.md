# Commerce Konnect Tunisia

Integrates the Konnect.network payment gateway with Drupal Commerce.

## Requirements
- Drupal Commerce 2.x
- A Konnect Merchant account (konnect.network)

## Installation
1. Enable the module.
2. Go to /admin/commerce/config/payment-gateways.
3. Add a new gateway of type "Konnect Tunisia".
4. Enter your API Key and Wallet ID.
5. Copy the "Webhook URL" displayed in the gateway settings and paste it into your Konnect Dashboard.
